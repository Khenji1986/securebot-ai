2. Welche Daten wir erheben

2.1 Bei Nutzung des Telegram-Bots

Bei der Nutzung von SecureBot AI werden folgende Daten verarbeitet:

  • Telegram User ID - Zur Identifikation deines Accounts
  • Telegram Benutzername - Fuer die Kommunikation und Abo-Zuordnung
  • Vorname - Fuer personalisierte Ansprache
  • Gestellte Fragen - Zur Beantwortung und Qualitaetssicherung
  • Generierte Antworten - Zur Dokumentation
  • Nutzungszeitpunkte - Fuer Nutzungsstatistiken und Limits
  • Abo-Status - Zur Verwaltung deines Plans (Free/Pro/Business)

2.2 Bei Nutzung des Phishing-Checkers

Wenn du den Phishing-Checker nutzt, werden die von dir gesendeten URLs und Links verarbeitet und analysiert. Diese URLs koennen personenbezogene Daten enthalten (z.B. E-Mail-Adressen in URL-Parametern). Die URLs werden ausschliesslich zur Analyse gespeichert und nicht an Dritte weitergegeben (ausser an Claude AI zur Analyse, siehe Abschnitt 4.2).

2.3 Bei Besuch der Webseite (Landing Page)

Unsere Webseite (gehostet auf GitHub Pages) erhebt:

  • Spracheinstellung - Wird lokal in deinem Browser gespeichert (localStorage). Diese Daten verlassen dein Geraet nicht.
  • Technische Zugriffsdaten - Durch den Hosting-Anbieter GitHub (siehe Abschnitt 4.5).

Wir setzen keine Cookies und nutzen keine Tracking- oder Analyse-Tools auf unserer Webseite.

4. Drittanbieter und Datenuebermittlung

4.1 Telegram

Wir nutzen die Telegram Bot API zur Kommunikation. Telegram verarbeitet deine Nachrichten gemaess deren Datenschutzrichtlinie. Telegram speichert Nachrichten auf ihren Servern (verschluesselt). Wir haben keinen Einfluss auf die Datenverarbeitung durch Telegram.

4.2 Anthropic (Claude AI)

Deine Fragen (inkl. URLs beim Phishing-Check) werden an Anthropic (USA) zur Verarbeitung durch Claude AI gesendet. Die Uebermittlung erfolgt auf Basis von Standardvertragsklauseln (SCCs) gemaess Art. 46 Abs. 2 lit. c DSGVO. Laut Anthropics API-Nutzungsbedingungen werden API-Daten nicht zum Training ihrer Modelle verwendet.

4.3 Stripe (nur bei Bezahl-Abos)

Bei Pro- oder Business-Abos werden Zahlungsdaten durch Stripe Inc. (USA) verarbeitet. Stripe ist PCI DSS Level 1 zertifiziert. Wir speichern keine Kreditkartendaten. Die Datenuebermittlung erfolgt auf Basis von SCCs. Mehr: Stripe Datenschutz

4.4 Google Cloud Platform (Hosting)

Der Bot laeuft auf Servern von Google Cloud in Frankfurt (europe-west3), Deutschland. Deine Daten verlassen die EU nur bei der Verarbeitung durch Claude AI (siehe 4.2) und ggf. Stripe (siehe 4.3).

4.5 GitHub Pages (Webseite)

Unsere Landing Page wird auf GitHub Pages (GitHub Inc. / Microsoft Corp., USA) gehostet. Beim Besuch der Webseite werden technische Zugriffsdaten (IP-Adresse, Browser-Typ, Zugriffszeitpunkt) durch GitHub verarbeitet. Mehr: GitHub Privacy Statement
