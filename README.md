## Was ist SecureBot AI?

Ein Telegram Bot der als AI-gestützter Security Berater fungiert.
Nutzer können Fragen zu Cybersecurity, IT-Sicherheit, DSGVO und mehr stellen.

## Features

- 24/7 verfügbar
- Powered by Claude AI (Anthropic)
- Spezialisiert auf IT-Security
- Mehrsprachig (DE/EN)
- Freemium Modell mit Stripe-Zahlung

## Tech Stack

- Python 3.11+
- python-telegram-bot
- Anthropic Claude API
- SQLite (User-Tracking)
- Stripe (Zahlungen)

## Für Lee: So startest du

1. API Keys in `.env` eintragen
2. `docker-compose up -d` ausführen
3. Fertig - Bot läuft!
